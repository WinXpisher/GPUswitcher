# **GPU switcher for Linux created with Qt**

---

**Description:**
This app was created to switch modes between Hybrid(Nvidia GPU + gCPU) and Integrated(gCPU). For this, you need supergfxctl.

## **Getting started**

1. Clone repository:
   ```bash
   git clone https://github.com/WinXpisher/GPUswitcher.git
2. Move to Ready catalog:
   ```bash
   cd GPUswitcher-main/Ready
3. Run the program:
   ```bash
   ./GPUswitcher.AppImage
